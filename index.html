jquery-oembed-all

A fork with improvements of the jquery-oembed project

This is a fork of the jquery-oembed located at http://code.google.com/p/jquery-oembed/. Instead of using oohembed or other such services it tries to embed the object natively without using a third-party server. This project will happily use native oembed services when it can, however it also use other types of embedding whenever oembed is not possible.
