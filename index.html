jquery-oembed-all by starfishmod

A fork with improvements of the jquery-oembed project

This is a fork of the jquery-oembed located at http://code.google.com/p/jquery-oembed/. Instead of using oohembed or other such services it tries to embed the object natively without using a third-party server. This project while it will happily use the the oembed framework it is not against using other types of embedding where ever possible.
